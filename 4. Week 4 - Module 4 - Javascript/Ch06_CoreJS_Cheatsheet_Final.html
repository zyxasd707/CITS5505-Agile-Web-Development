12. Prototype 原型与原型链 (P50–54) ★★

P50: JS不用class→原型式prototype-based(无类/面向原型/基于实例)

// 最佳: 方法放prototype上→所有实例共享!(P51)
function Drink(p){this.price=p;}       // 数据在实例
Drink.prototype.get=function(){        // 方法在prototype
  return this.price*1.125;};
let a=new Drink(5),b=new Drink(6);
a.get===b.get // true! 共享同一函数

14. Closures 闭包 (P62–64) ★★

P62: 函数内可声明函数; 内部函数访问外层变量; 闭包=函数的局部变量在函数返回后仍存活

// 课件经典: makeAdder (P63)
function makeAdder(a){        // a被闭包捕获
  return function(b){return a+b;}; // 返回的函数"记住"a
}
let add5=makeAdder(5);  // scope:{a:5}
let add20=makeAdder(20);// scope:{a:20} 独立!
add5(6)→11  add20(7)→27 // 互不干扰!
  • 每次调用→新scope object保存局部变量(P63); 返回函数持有引用→不被GC(P64)
  • vs全局(P64): ①每次调用创建新scope ②scope不可从代码直接访问
Q: for(var i=0;i<3;i++){setTimeout(()=>console.log(i),100);} 输出?
A: 3,3,3。var函数级→一个i→循环结束i=3→都读3。改var为let→0,1,2(每次独立)
