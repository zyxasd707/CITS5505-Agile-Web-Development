jQuery Deferred / Promise 链式回调

  • • $.get() 返回 jqXHR — 同时是 Promise
  • • .done() → 200 OK & 有响应体
  • • .fail() → 网络错误 / 4xx / 5xx
  • • .always() → 无论成败都执行
  • • 链式写法避免回调金字塔

Dog CEO Public API

  • • 端点: /api/breeds/image/random
  • • 方法: GET · 无需 API key
  • • 响应: {message: URL, status: "success"}
  • • 支持 CORS → 浏览器可直连
  • • 同类: PokéAPI / JSONPlaceholder / Open-Meteo

为什么"不冻结 UI"

  • • XHR/Fetch 走浏览器网络线程
  • • 主线程可继续处理事件循环
  • • loading 态 + .always 恢复按钮
  • • 防重入: $btn.disabled=true
  • • P21 要点: Does not freeze env
