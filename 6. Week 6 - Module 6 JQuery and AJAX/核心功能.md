**jQuery** 是一个快速、简洁且功能丰富的 **JavaScript 函数库**。它的设计理念是 "Write Less, Do More"（写得更少，做得更多），旨在简化 HTML 文档遍历、事件处理、动画设计和 Ajax 交互。 [1, 2, 3, 4] 

## 核心功能

- **DOM 操作**：通过类似 CSS 选择器的语法，极其简便地选取、修改或删除 HTML 元素。
- **跨浏览器兼容**：它统一了不同浏览器之间的 API 差异，让开发者无需担心代码在 Chrome 或 Safari 下的表现不同。
- **事件处理**：简化了点击、悬停、提交等用户交互的监听代码。
- **动画效果**：内置了许多简单的动画函数，如隐藏、淡入淡出和滑动。
- **Ajax 支持**：让异步数据请求（如从服务器获取数据而不刷新页面）变得非常简单。 [2, 3, 4, 5, 6, 7]

## jQuery 示例：点击按钮隐藏文本

在使用 jQuery 之前，你需要先引入它的库文件（通常通过 CDN 引入）。

```html
<!DOCTYPE html>
<html>
<head>
    <title>jQuery示例</title>
    <script src="https://code.jquery.com/jquery-3.6.0.min.js"></script>
    <style>
        body { font-family: Arial, sans-serif; padding: 20px; }
        .box { width: 200px; height: 100px; background: skyblue; margin: 10px 0; padding: 10px; }
        .active { background: coral; }
    </style>
</head>
<body>
    <h2>jQuery示例</h2>
    
    <input type="text" id="inputText" placeholder="输入文字">
    <button id="addBtn">添加</button>
    <button id="clearBtn">清空</button>
    
    <div class="box">点击我切换颜色</div>
    <div class="box">我也可以被点击</div>
    
    <ul id="myList">
        <li>项目 1</li>
        <li>项目 2</li>
    </ul>

    <script>
        $(document).ready(function(){
            // 1. 点击切换样式
            $(".box").click(function(){
                $(this).toggleClass("active");
            });

            // 2. 添加新项
            $("#addBtn").click(function(){
                var text = $("#inputText").val();
                if(text){
                    $("#myList").append("<li>" + text + "</li>");
                    $("#inputText").val("");
                }
            });

            // 3. 清空列表
            $("#clearBtn").click(function(){
                $("#myList").empty();
            });

            // 4. 鼠标悬停效果
            $("li").hover(
                function(){ $(this).css("background", "lightgray"); },
                function(){ $(this).css("background", "white"); }
            );
        });
    </script>
</body>
</html>
```

## 语法解析

1. **`$` 符号**：jQuery 的别名，是访问 jQuery 函数库的入口。
2. **选择器 `$("#btn")`**：这会选取 HTML 中 `id="btn"` 的元素，语法与 CSS 完全一致。
3. **`.click(function(){...})`**：为选中的元素绑定点击事件。
4. **`.hide()`**：jQuery 内置的动画方法，用于改变元素的 CSS 属性使其隐藏。 [2, 4, 6, 8]
