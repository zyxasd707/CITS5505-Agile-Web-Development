P9 异步通信交互图

Sequence Diagram — Asynchronous Communication with jQuery Callback Pattern

Flow A: 异步请求流程

  • • 用户点击 fetchBtn 发起请求
  • • big_Request() 调用 $.get() 发送异步 HTTP
  • • 主线程立即空闲，不等待响应
  • • 响应到达后 .done() 触发回调
  • • callback(error, data) 模式处理结果

Flow B: 非阻塞证明

  • • 等待 $.get 响应期间点击 countBtn
  • • 计数器正常 +1 更新
  • • 证明 UI 没有被冻结
  • • JS 单线程，但运行环境不是单线程
  • • 网络请求由浏览器线程处理

P9 回调模式

  • • 模式: Object.asyncFn(params, callbackFn)
  • • 回调签名: function(error, response)
  • • error 非空 → 走错误分支
  • • error 为 null → 走 render(data) 分支
